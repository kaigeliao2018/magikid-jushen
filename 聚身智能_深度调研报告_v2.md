## 一、公司全景

### 1.1 基本信息
| 项目 | 详情 |
|------|------|
| **公司全称** | 聚身智能（EMi® / AIMAN） |
| **总部地点** | 中国杭州 |
| **企业定位** | 具身智能生态链企业 |
| **官方网站** | https://www.aiman.world |
| **核心理念** | 解放异化劳动 |
| **成立时间** | 约2024-2025年（新兴企业） |

### 1.2 三大业务板块

#### ① 聚身智造（代工生产）
- **核心能力**：OEM/ODM代工、规模化量产
- **基础设施**：8000㎡组装直播考试基地
- **产品范围**：
  - 人形机器人整机
  - 机械臂/灵巧手
  - 轮式/轮臂复合机器人
  - 核心零部件

#### ② 采了么APP（数据平台）
- **定位**：AI数据采集操作平台（"机器人的ImageNet"）
- **模式**：劳动即收入（众包模式）
- **目标**：构建1000万劳动者社区
- **服务**：机器人训练数据众包采集

#### ③ 聚身之家OPC（渠道平台）
- **功能**：挑本体、找专家、渠道价
- **合作方**：星踪岛（战略合作伙伴）
- **运营**：与杭州市场景创新中心深度合作
